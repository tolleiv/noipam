Example usage

Check if IP is in use:

curl -X GET    -s http://{{ base_uri }}/ip/10.20.70.11

Mark IP as used/unused

curl -X PUT    -s http://{{ base_uri }}/ip/10.20.70.11
curl -X DELETE -s http://{{ base_uri }}/ip/10.20.70.11

Get an available IP from a subnet

curl -X POST   -s http://{{ base_uri }}/net/next/10.20.70.0/24

List the remaining or used IPs for a subnet

curl -X GET     -s http://{{ base_uri }}/net/remaining/10.20.70.0/24
curl -X GET     -s http://{{ base_uri }}/net/used/10.20.70.0/24

List all connections for one IP address

curl -X GET    -s http://{{ base_uri }}/connection/10.9.1.3

List all connections for a network

curl -X GET    -s http://{{ base_uri }}/net/connected/10.9.1.0/24

Add a "connection" relation between two addresses

curl -X PUT    -s http://{{ base_uri }}/connection/10.9.1.3/to/2.2.2
curl -X DELETE -s http://{{ base_uri }}/connection/10.9.1.3/to/2.2.2
